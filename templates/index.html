About This Project

The Academic FAQ Chatbot is an advanced AI-powered assistant designed to help students, faculty, and staff quickly find answers to academic questions. Built with cutting-edge natural language processing and semantic search technology, it provides instant, accurate responses with proper citations.

This project combines Google Gemini's powerful language model with FAISS vector search to understand context and retrieve the most relevant information from academic handbooks, policies, and guidelines.
